# 邻宠到家视频视觉规范

> 本视觉规范用于作品集宣传视频，不代表真实品牌规范。

## 设计关键词

- 温暖
- 干净
- 社区服务感
- 可信
- 轻量科技感

## 色彩

| 用途 | 色彩建议 | 说明 |
| --- | --- | --- |
| 主色 | 青绿色 / 深绿色 | 表达可信、稳定、社区服务 |
| 背景 | 浅米色 / 浅绿色 | 保持温暖和亲近感 |
| 强调 | 暖黄色 / 橙色 | 用于 CTA 和关键提示 |
| 文本 | 深灰绿 | 保持可读性 |

## 字体与排版

- 字幕短句化，每屏只强调一个信息点。
- 标题优先使用较粗字重，正文保持克制。
- 避免堆叠过多解释性文字。
- 重要数字如价格、评分、服务次数可放大显示。

## 组件语言

- 服务卡片：服务名、场景、价格起点。
- 照护者卡片：头像、姓名、片区、评分、服务次数、擅长类型。
- 反馈卡片：照片、状态、饮水/活动/情绪、建议。
- CTA：预约咨询、查看服务、先了解照护者。

## Do

- 用画面承接用户焦虑，再用服务和反馈缓解焦虑。
- 保持和小程序/H5 的卡片式视觉一致。
- 明确突出“附近、可信、反馈”三个关键词。
- 让每一屏都能被手机用户快速读完。

## Don't

- 不使用夸张医疗、安全或赔付承诺。
- 不把作品集模拟素材包装成真实投放数据。
- 不塞入太多业务细节，视频只负责第一层理解。
- 不用过强营销话术压过产品可信感。

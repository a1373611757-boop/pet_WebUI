# 邻宠到家 30 秒竖版品牌介绍视频

> 本视频为作品集传播物料，画面、脚本、数据和服务案例均为模拟展示，不代表真实商业投放或真实用户评价。

## 视频定位

`neighbor-pet-home-promo/` 用来展示邻宠到家的品牌传播表达能力：如何把宠物照护服务的核心卖点，在 30 秒竖版视频中讲清楚。它适合微信视频号、朋友圈、小红书等移动端场景。

## 传播目标

- 让用户快速理解“外出时宠物也有人认真照顾”。
- 传达社区附近、可信照护、过程反馈、价格清晰的核心卖点。
- 为 H5 或小程序预约入口制造第一层兴趣。
- 在作品集中展示候选人对品牌表达、脚本和分镜的组织能力。

## 目标受众

- 宠物主人：加班、出差、旅行、临时外出前需要安排照护。
- 面试官：评估候选人是否能把产品卖点转化为传播叙事。

## 核心情绪

- 开头：担心和临时焦虑。
- 中段：看到附近照护者和服务流程后逐渐安心。
- 结尾：清楚知道可以预约和获得反馈。

## 与产品文档的关系

- 卖点来自 PRD：可信照护者、按需预约、过程反馈。
- 场景来自用户旅程：临时外出、第一次托付、服务反馈。
- 风险边界保持一致：不承诺绝对安全，不包装真实用户案例。

## 面试讲解重点

视频不是单独的视觉作品，而是产品传播链路的一部分：它负责把 PRD 中的用户痛点和服务卖点转化为用户能快速理解的情绪叙事。

# 邻宠到家 H5 Demo

> 本 H5 Demo 为作品集展示材料，页面中的服务说明、价格、图片和案例用于演示，不代表真实商业服务。

## 定位

`04-H5-Demo/` 是邻宠到家的用户侧服务介绍原型，用来展示宠物主人在提交预约前如何理解服务类型、信任机制、照护流程和行动入口。它不是普通 Vite 模板，也不承担真实获客转化。

## 解决的问题

- 用户不知道自己应该选择上门喂猫、上门遛狗还是日托。
- 用户第一次托付宠物时，需要看到可信照护者、服务流程和反馈机制。
- 用户需要在手机页面中快速理解价格、流程和下一步操作。

## 页面应表达的核心信息

- 品牌定位：社区宠物照护服务。
- 服务范围：上门喂猫、上门遛狗、宠物日托/寄养。
- 信任机制：附近照护者、评分、服务次数、过程反馈。
- 预约路径：选择服务 -> 填写需求 -> 确认照护者 -> 查看订单/反馈。
- 边界声明：当前为作品集模拟 Demo，不接真实下单和支付。

## 面试讲解重点

H5 Demo 展示的是“用户第一次理解服务”的入口体验。面试时可以用它说明：我不是直接让用户填表，而是先处理服务选择、信任建立和流程预期，再引导到小程序预约。

# 邻宠到家｜产品经理求职作品集

> 本仓库是求职作品集，不是真实上线业务。经营数据、订单、用户案例、AI 输出均为模拟内容；项目不接真实后端、不接微信支付、不包含真实用户数据。

## 1 分钟看懂

邻宠到家是一个社区宠物服务预约项目，围绕宠物寄养、上门喂猫、上门遛狗和日托场景，展示从产品定义、用户流程、小程序 MVP、经营数据看板到 AI 功能设计的完整产品经理工作样本。

核心问题：

- 宠物主人临时外出时，很难快速找到可信照护者。
- 托付宠物的信任成本高，用户需要看到身份、经验、评价和服务过程反馈。
- 平台运营需要理解渠道转化、服务供给、入住率和复购，而不只是展示订单列表。
- AI 需要嵌入客服、日报、文案和经营复盘，而不是做一个泛聊天入口。

## 文档结构

`docs/` 按产品经理工作流组织：

1. 产品定位与策略
2. 市场与行业机会
3. 问题定义与用户洞察
4. 用户旅程地图
5. 机会方案树
6. PRD
7. MVP 用户故事地图
8. 指标体系与数据看板
9. AI 功能方案
10. 优先级与路线图
11. 作品集边界声明
12. 面试讲解稿

## 适合展示的岗位能力

- 产品经理：问题定义、MVP 范围、用户场景、故事地图、路线图。
- AI 产品助理：AI 客服、AI 日报、AI 文案、AI 经营分析的场景设计和风险边界。
- 数据分析师：指标口径、漏斗、渠道转化、入住率、复购和业务解释。
- AI 解决方案顾问：从行业问题到产品流程、AI 模块、技术边界和落地路径的完整表达。

## 本地预览

- 微信小程序：用微信开发者工具打开本目录，编译 `miniprogram/`。
- 看板：进入 `NeighborPet-Portfolio/dashboard/` 后运行 `npm install` 和 `npm run dev`。
- H5 Demo：进入 `04-H5-Demo/` 后运行 `npm install` 和 `npm run dev`。
- 视频项目：查看 `neighbor-pet-home-promo/neighbor-pet-home-promo-9x16.mp4` 与对应脚本/分镜。

## 边界声明

本作品集不包含真实登录、真实支付、真实客服、真实地图调度、真实 AI API 调用或真实宠物照护履约。所有内容用于展示产品思考、原型实现、指标设计和 AI 应用设计能力。

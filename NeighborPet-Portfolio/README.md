# NeighborPet Portfolio

> 本目录是邻宠到家作品集的案例页与展示工程集合。所有经营数据、用户案例和 AI 输出均为模拟内容。

## 目录定位

`NeighborPet-Portfolio/` 承载项目的深度展示材料，包括核心 PRD、经营数据看板、AI 功能演示中心和可部署的案例页。它面向面试官，目标是让对方快速判断候选人的产品、数据和 AI 应用设计能力。

## Project Narrative

邻宠到家围绕宠物寄养与上门照护场景，拆解了三个核心问题：

1. 宠物主人如何在临时外出时快速找到可信照护者。
2. 服务过程如何通过记录、照片和日报建立透明度。
3. 运营人员如何通过渠道、漏斗、入住率、复购和 AI 洞察做经营复盘。

## 主要成果

- `docs/PRD-邻宠到家.md`：核心 PRD，定义用户、场景、MVP、指标和边界。
- `dashboard/`：React + TypeScript 数据看板与 AI 功能演示中心。
- `dashboard/docs/AI功能方案.md`：AI 客服、日报、文案、经营分析的产品方案。
- `dashboard/public/docs/`：已发布展示材料入口。

## Quick Start

```bash
cd dashboard
npm install
npm run dev
```

## 与根目录文档的关系

根目录 `docs/` 是求职作品集标准化文档体系；本目录保留案例页工程和原始展示材料。推荐面试时先从根目录 `README.md` 或 `docs/00-作品集总览.md` 进入，再根据岗位关注点打开这里的 PRD 和看板。
